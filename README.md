# Vivi Apology Slideshow
Make sure to have nodejs installed <3


A romantic draggable slideshow built with React, Vite, Tailwind, Framer Motion, Three.js, and lucide icons.

## Run locally

```bash
npm install
npm run dev
```

## Deploy to Vercel

Drag this folder into Vercel, or import it as a Git project. Vercel should detect Vite automatically.

Build command:

```bash
npm run build
```

Output directory:

```bash
dist
```
